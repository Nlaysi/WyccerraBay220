code/_helpers/time.dm

/proc/Uptime Real time since the server started. Same concept as REALTIMEOFDAY.
/proc/time_to_readable Converts an integer of world.time to a user-readable string split into time measurements from seconds to years.
/proc/start_watch Returns "watch handle" (really just a timestamp :V)
/proc/stop_watch Returns number of seconds elapsed. @param wh number The "Watch Handle" from start_watch(). (timestamp)
