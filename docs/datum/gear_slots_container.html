Space Station 13 - Modules - Types — Var Details - Proc Details

gear_slots_container
/datum/gear_slots_container

Simple objects that represents all gear slots the player has in preferences.

Vars

gear_slotsThe flat list of gear slots, where index of slot in the list is slot number
picked_gear_slotThe gear slot that is currently displayed in preferences
sizeMax size of the container. New gear slots are added to gear_slots when required

Procs

cycle_slot_leftSets picked_gear_slot to the picked_gear_slot - 1, or to length(gear_slots) if new slot is <= 0 Returns true, if slot successfully changed, false otherwise
cycle_slot_rightSets picked_gear_slot to the picked_gear_slot + 1, or to 1 if overflowed Returns true, if slot successfully changed, false otherwise
fill_with_slotsCreate new /datum/gear_slot until gear_slots length wil be equal to size. If the gear_slots is already of desired size, no changes will happen
get_gear_slotsGet copy of get_gear_slots list
get_picked_gear_slotGet currently picked gear slot. If no valid slot picked, we pick the first one
get_picked_gear_slot_numberGet currently picked gear slot number.
get_sizeReturs the size of the container
set_sizeTrims gear_slots to desired size. trim_size must be greater or equal to 0.
